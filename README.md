# 音樂殿堂 - 樂器行網站

一個現代化、響應式的樂器行網站，使用純 HTML、CSS 和 JavaScript 建立。

## 功能特色

- 🎸 響應式設計，支援各種裝置
- 🎹 商品展示與分類
- 🥁 互動式購物車功能
- 🎤 聯絡表單
- ✨ 流暢的動畫效果

## 線上展示

訪問網站：[https://YOUR-USERNAME.github.io/music-store/](https://YOUR-USERNAME.github.io/music-store/)

## 本地開發

1. Clone 此專案
```bash
git clone https://github.com/YOUR-USERNAME/music-store.git
```

2. 使用任何 HTTP 伺服器運行
```bash
# 使用 Python
python3 -m http.server 8000

# 或使用 Node.js
npx http-server
```

3. 在瀏覽器開啟 `http://localhost:8000`

## 技術棧

- HTML5
- CSS3 (含動畫和 Flexbox/Grid)
- Vanilla JavaScript
- Font Awesome 圖示
